# JBE 매니저 v1.4.2 사용자 매뉴얼

본 매뉴얼은 JBE 축구동호회의 통합 관리 시스템(JBE Manager)의 운영 방법을 설명합니다.

---

## 🚀 1. 시스템 개요
이 시스템은 구글 스프레드시트(Google Sheets)와 앱스 스크립트(GAS)를 기반으로 작동하며, 회원 정보, 출석, 팀 배정, 자동화 업무를 처리합니다.

*   **웹 대시보드 주소**: `https://jbe-manage.pages.dev` (회원 목록 열람용)
*   **관리자 페이지**: 구글 스프레드시트 (데이터 입력 및 관리용)

---

## 📋 2. 회원 관리 (`회원명단` 시트)
회원 정보는 반드시 **`회원명단`** 시트에서 관리해야 합니다. (구 `등록부` 사용 금지)

### 2.1 신규 회원 추가
1.  **`회원명단`** 시트의 가장 아래 빈 행에 정보를 입력합니다.
2.  **성명(필수)**, 등번호, 포지션, 직급 등을 기입합니다.
3.  **회원번호(A열)**는 비워두어도 됩니다.
    *   *Tip: 나중에 스크립트 실행 시 자동으로 채워지거나, 필요하면 'M001' 형식으로 직접 입력해도 됩니다.*
4.  **상태(I열)**는 기본적으로 '활동'으로 입력해주세요.

### 2.2 정보 수정
*   이름 개명, 등번호 변경 등은 시트에서 직접 수정하면 됩니다.
*   대시보드에는 **약 5~10분 이내**에 자동으로 반영됩니다. (즉시 반영을 원하면 GAS 재배포 필요)

---

## 📅 3. 출석 체크 및 상태 관리

### 3.1 자동 상태 업데이트 (활동/휴면)
매일 **새벽 4시**에 시스템이 자동으로 다음 기준에 따라 회원의 상태를 갱신합니다.
*   **활동**: 최근 3개월 이내 출석 기록이 1회 이상 있는 회원.
*   **휴면**: 최근 3개월간 출석이 없는 회원. (대시보드에서 회색 배지)
*   **장기휴면**: 1년 이상 출석이 없는 회원. (대시보드에서 붉은색 배지)

### 3.2 수동 출석 체크 (`출석부` 시트)
1.  **`출석부_2026`** 시트로 이동합니다.
2.  해당 날짜 열(예: 3월 1일)을 찾습니다.
3.  참석한 회원의 셀에 숫자 **`1`**을 입력합니다.
    *   입력 시 자동으로 셀 색상이 초록색으로 변합니다.

## ⚙️ 4. 관리자 기능 (Apps Script)
구글 스프레드시트 상단 메뉴바의 **[확장 프로그램] > [Apps Script]**에서 실행하거나, 시트 내 버튼(향후 추가 시)을 통해 실행합니다.

| 함수명 | 설명 |
| :--- | :--- |
| **`runSystemCheck`** | 시스템 건강검진 기능. 회원등록/출석/팀배정 기능을 테스트합니다. |
| **`updateMemberStatus`** | 강제로 회원 상태(활동/휴면)를 최신화합니다. (새벽까지 기다리기 싫을 때 사용) |
| **`testEmail`** | 에러 알림 메일이 잘 오는지 테스트합니다. |
| **`setupTriggers`** | 자동화(새벽 4시 실행)가 풀렸을 때 다시 설정하는 함수입니다. |

## ⚠️ 5. 문제 해결 (Troubleshooting)

### Q1. 대시보드에 아무것도 안 떠요.
*   **원인**: 브라우저가 옛날 버전을 기억하고 있거나, GAS 배포가 꼬인 경우입니다.
*   **해결**:
    1.  브라우저에서 **`Ctrl + F5`** (강력 새로고침)를 눌러보세요.
    2.  그래도 안 되면 개발자에게 "GAS 배포 URL 확인"을 요청하세요.

### Q2. 분명히 오늘 출석 체크 했는데 '휴면'이래요.
*   **원인**: 자동화 스크립트는 **매일 새벽 4시**에 한 번만 돕니다.
*   **해결**: 내일 아침이 되면 '활동'으로 바뀌어 있을 겁니다. 당장 바꾸고 싶다면 `updateMemberStatus`를 수동 실행하거나, 시트에서 직접 '활동'으로 고치셔도 됩니다.

### Q3. 에러 알림 메일이 왔어요.
*   **`ScriptError`**: 코드에 문제가 있는 것입니다. 메일 내용을 캡쳐해서 개발자에게 보내주세요.
*   **`NetworkError`** / **`Band API Error`**: 일시적인 인터넷 문제일 수 있습니다. 잠시 후 다시 시도해보세요.
