# JBE 매니저 (Soccer Club Management System)

JBE 축구동호회를 위한 회원 관리, 출석 체크, 팀 배정 자동화 시스템입니다.

## 📁 폴더 구조

- `gas/`: Google Apps Script 소스 코드 (회원관리, 출석체크 등)
- `dashboard/`: 관리자 대시보드 웹페이지 소스 (GitHub Pages/Cloudflare Pages 배포용)
- `docs/`: 사용자 가이드 및 개발 문서

## 🚀 주요 기능

1. **회원 관리**: 구글 폼 연동 신규 회원 자동 등록, 등번호 관리
2. **출석 관리**: 네이버 밴드 투표 결과 파싱 및 자동 출석 체크
3. **팀 배정**: 포지션 및 실력 기반 자동 팀 밸런싱
4. **알림**: 경기 일정 및 결과 자동 포스팅

## 🛠 사용 기술

- **Backend**: Google Apps Script (GAS), Google Sheets
- **Frontend**: HTML5, CSS3, Vanilla JS (Dashboard)
- **Deployment**: GitHub, Cloudflare Pages
