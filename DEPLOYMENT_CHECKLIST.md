## 🆘 문제 발생 시 대응

### 1. 트리거 미작동
```javascript
// Apps Script에서 실행
setupTriggers();
```

### 2. 데이터 손실
- `Log` 시트에서 작업 이력 확인
- `*_Backup_*` 시트에서 복구

### 3. 대시보드 오류
- 브라우저 캐시 삭제 (Ctrl+F5)
- GAS Web App 재배포
- `script.js`의 URL 확인
